# Gamepad Control

## Supported Controllers

- Xbox One / Series X|S
- PlayStation DualShock 4 / DualSense
- Generic USB gamepads

## Control Mapping

| Input | Action |
|-------|--------|
| Left Stick | Move direction |
| Right Stick | Turn / Look |
| Left Trigger | Decrease speed |
| Right Trigger | Increase speed |
| A / Cross | Start |
| B / Circle | Stop |
| D-Pad | Select movement style |
