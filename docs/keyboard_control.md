# Keyboard Control

## Key Bindings

| Key | Action |
|-----|--------|
| W | Move forward |
| S | Move backward |
| A | Strafe left |
| D | Strafe right |
| Q | Turn left |
| E | Turn right |
| Space | Stop |
| Up Arrow | Increase speed |
| Down Arrow | Decrease speed |
| 1-9 | Select movement style |

## Movement Styles

1. Walk (default)
2. Run
3. Stealth
4. Happy
5. Injured
6. Kneeling
7. Hand Crawling
8. Elbow Crawling
9. Boxing
